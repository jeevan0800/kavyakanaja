# kavyakanaja# Kavya Kanaja (ಕಾವ್ಯ ಕಣಜ)

Kavya Kanaja is an Android application dedicated to Kannada literature. It offers a rich collection of Kannada poems, daily poem recommendations, and an extensive library featuring the biographies and works of prominent Kannada poets.

## Features

*   **Daily Poem (ದಿನದ ಕವನ):** A curated daily poem retrieval system powered by a 500-poem JSON database.
*   **Text-to-Speech (ಓದು):** Integrated text-to-speech functionality that allows users to listen to poem recitations.
*   **Poet's Corner (ಕವಿಗಳ ಪರಿಚಯ):** A comprehensive library featuring the biographies and selected works of over 30 prominent Kannada poets.
*   **Modern UI:** Built entirely with Jetpack Compose for a smooth and intuitive user experience.

## Tech Stack

*   **Language:** [Kotlin](https://kotlinlang.org/)
*   **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
*   **Navigation:** Jetpack Navigation Compose
*   **Data Parsing:** Gson

## Getting Started

### Prerequisites

*   [Android Studio](https://developer.android.com/studio) (latest version recommended)
*   Android SDK (Minimum API level 24, Target API level 34)

### Installation

1.  Clone the repository or download the source code.
2.  Open the project in Android Studio.
3.  Let Gradle sync the project dependencies.
4.  Run the application on an Android emulator or a physical device.
